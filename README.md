# HTML Resume Project

## Project Overview
This project is a simple and professional resume created using only HTML.
It focuses on proper structure, semantic HTML tags, and clear readability.

## Technologies Used
- HTML5

## Setup Steps
1. Clone the GitHub repository
2. Open the project folder
3. Open the `index.html` file in any web browser

No additional setup or installation is required.

## Usage
- Open the resume in a browser
- View sections such as Projects, Education, and Contact
- The resume is clean, readable, and easy to understand
